
# Membuat Multi Languages dengan Next.js

Kali ini saya membuat modul Multi Languages atau Multi Bahasa dengan framework Next.js. Let's start running this code!

## Quick Start
``` bash
$ yarn

$ yarn dev
```
Kemudian akses http://localhost:3000 di browser.
